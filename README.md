## Overview

VizoThink is your AI-powered thought partner. Capture ideas, explore connections, and let the machine help you think through the chaos.

**⚠️ WORK IN PROGRESS** - This project is under active development.

## Features

- 💡 **Idea Capture** - Dump thoughts instantly. Auto-organize with AI clustering.
- 🧠 **Mind Maps** - Visualize thought connections. See the big picture.
- 🔥 **AI Spark** - Get instant insights. Let AI challenge your assumptions.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **AI:** Local LLM Integration (Upcoming)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```
